## üöÄ Features
The system is a menu-driven application that allows a bank officer to:
1. **Add New Customers:** Register up to 5 savings account holders with personal details.
2. **View Specific Customer:** Search for a customer using their unique account number.
3. **View All Customers:** Display a summary list of all registered accounts and balances.
4. **Deposit Money:** Add funds to an existing account.
5. **Withdraw Money:** Remove funds from an account (with balance updates).
6. **Update Details:** Modify customer information (NIC, Name, Address, etc.) without changing the account number.
7. **System Exit:** Securely exit the application.
